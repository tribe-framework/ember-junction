# ember-junction

## Compatibility

* ember-tribe 2.x.x

## Installation

```
ember init; yes | ember install ember-tribe; yes | ember install ember-junction;
```
Then fill in the .env file with Tribe credentials and run:
```
php sync-types.php
```
