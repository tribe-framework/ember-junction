# ember-junction

## Compatibility

- ember-tribe 2.x.x

## Installation

```
ember init; yes | ember install ember-tribe; yes | ember install ember-junction;
```

Then fill in the .env file with Tribe API credentials.

```
TRIBE_API_URL=""
JUNCTION_PASSWORD=""
JUNCTION_SLUG="junction"
```
